IDEA DEL PROYECTO
Convive es una aplicación web diseñada para facilitar la gestión económica y la organización en pisos compartidos.
Muchas personas que viven con compañeros tienen problemas para gestionar gastos comunes como alquiler, facturas, compras del supermercado o reparaciones. Esto puede generar confusión o conflictos si no existe una forma clara de controlar quién ha pagado qué o cuánto debe cada persona.
Convive busca solucionar este problema ofreciendo una plataforma donde todos los miembros del piso puedan registrar gastos, dividirlos automáticamente, ver sus deudas y pagos pendientes, y mantener un control transparente de la economía del hogar.
Además de la gestión financiera, la aplicación también ofrece herramientas para mejorar la organización del piso, como calendario de pagos, gestión de tareas o sistema de notificaciones.

NUESTROS OBJETIVOS
Los principales objetivos del proyecto son:
Facilitar la gestión de gastos compartidos entre compañeros de piso.
Evitar conflictos relacionados con el dinero mediante transparencia en las cuentas.
Automatizar el cálculo de deudas entre los miembros del piso.
Centralizar la gestión de facturas, compras y gastos comunes.
Mejorar la organización del piso compartido.
Ofrecer un sistema de recordatorios y notificaciones de pago.
Mostrar de forma clara el balance económico individual y grupal.

FUNCIONALIDADES WEB

La aplicación contará con varios módulos principales:

- Sistema de Autenticación
Registro de usuarios
Inicio de sesión
Recuperación de contraseña
Creación de un piso o unión mediante código de invitación

- Panel de Control
Pantalla principal tras iniciar sesión donde el usuario puede ver:
Resumen económico del piso
Deudas pendientes
Próximos pagos
Accesos rápidos a las secciones principales

- Área Personal
Cada usuario tendrá su propio espacio donde podrá consultar:
Mis deudas: dinero que debe a otros compañeros
Me deben: dinero que otros miembros le deben
Historial de pagos
Resumen mensual de gastos
Calendario personal de pagos

- Área Grupal
Sección donde todos los miembros pueden ver información global del piso:
Resumen de gastos del piso
Balance entre compañeros
Optimización de pagos (calcular las transferencias mínimas necesarias)
Simulación de cambios en el piso (entrada o salida de miembros)

- Gestión de Gastos
Permite registrar gastos compartidos:
Subir tickets de compra
Dividir gastos entre los miembros del piso
Registrar imprevistos o reparaciones
Lista de la compra colaborativa

- Gestión de Facturas
Gestión de gastos recurrentes como:
Luz/Agua/Internet/Alquiler/Suscripciones
Incluye:
Subida de facturas
División del gasto entre miembros
Historial de facturas
Recordatorios automáticos de pago

 - Calendario
Sistema de calendario con:
Fechas de vencimiento de facturas
Registro de pagos
Organización de tareas del piso

- Sistema de Notificaciones
La aplicación enviará avisos cuando:
Se registre un nuevo gasto
Una factura esté próxima a vencer
Un compañero confirme un pago
Se realicen cambios en el piso
